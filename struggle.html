Characters

Main Character

This person is the main character. It does not have a name. Gender is unknown.

There are reasons why the main character looks like this.

  • Color White Represents the character is depressed.
  • Glasses represent shield of vulnerability (or emotion).
  • Eyes have no light. This represents lifeless.
  • Hair is blonde because it represents there is still hope.
  • The Shadows on its face represents loneliness and fatigue.
  • This character is nothing more than an empty husk.

Phone

Main Character holding a phone.

This tells the main character where it has to go at this time.

It was modeled after the phone I have.

FEAR

She represents Fear. Prevents the main character to ask for help. Messing around with the main character's vision. A mental state of the Main Character to never overcome.

Time-Management Character

This person represents time-management. Working long hours just to feel more depressed.

It can also represent not getting much school work done, and being used just to have their own goals done.
